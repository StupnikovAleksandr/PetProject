План тексирования :
# 1. Введение
Цель - покрыть автотестами функционал мобильного приложения


# 2. Объект тестирования :
Мобильное приложение V Hospice

Входит в тестирование:
UI приложения
Авторизация
Работа с сетью (API)
список
фильтрация
создание/удаление записей)

## ПОДУМАТЬ ЕЩЁ

Не входит:
Тестирование прозводительности
Локализация
Тестирование бекнд


# 3. Стратегия тестирования

Будут выполнены:
Функциональное тестирование
Интеграционное
UI-тестирование
Проверка ошибок и нестандартных сценариев
Не будут:
Нагрузочное тестирование


# 4. Среда тестирования

Устройства:
Android device API 29 (ПОКА 36)

Эмуляторы:
Pixel 4 (Android 16)
Инструменты:
Android Studio
ADB
Charles / Fiddler (опционально)


#  5. Риски :
Нестабильная работа сервера
Отсутсувие ТЗ
Запуск автотестов одновременно (при учёте что сервер у нас 1)
